## Raw Datasets

All datasets originate from the Kaggle repository:

Air Quality Data in India  
https://www.kaggle.com/datasets/rohanrao/air-quality-data-in-india

### Included in this repository
- `city_day.csv` — Daily city-level air quality data (used in Phase 1–4)

### Not included due to size
- `city_hour.csv`
- `station_day.csv`
- `station_hour.csv`

These datasets can be downloaded from the Kaggle link above and
are planned for use in future phases of this project.
